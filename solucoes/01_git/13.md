13 - Qual o comando para ver as diferenças entre o penúltimo _commit_ e 
o antepenúltimo _commit_? Podes assumir que estamos num ramo sem divergências, ou seja, cada _commit_ tem apenas um pai.

## Soluções

### Solução 1

O comando para visualizar estas diferenças é o seguinte:

```
$ git diff HEAD~1..HEAD~2
```

_Nota: Após git 1.8.5+ é possível trocar @ por HEAD ficando o comando da seguinte forma:_

```$ git diff @~1..@~2```

_Nota 2: Depois de mostrar as diferenças, para voltar a linha de comandos
"normal" será necessário premir a tecla ```Q```

*Solução proposta por [André Santos](https://github.com/Snigy24)*
